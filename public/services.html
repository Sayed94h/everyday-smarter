Baby Name API

Get 1000 names of baby girl and 1000 names of baby boy by using our Baby Name API

How to use this API?

1. Get all 1000 names of baby boy:
https://everydaysmarter.herokuapp.com/api/babyNames/boy
2. Get all 1000 names of baby girl:
https://everydaysmarter.herokuapp.com/api/babyNames/girl
3. Get one name of baby girl by Id:
https://everydaysmarter.herokuapp.com/api/babyNames/girl/a number(1-1000)
4. Get one name of baby boy by Id:
https://everydaysmarter.herokuapp.com/api/babyNames/boy/a number(1-1000)
5. Get all 1000 names of baby girl and sort them ascending:
https://everydaysmarter.herokuapp.com/api/babyNames/girl?sort=ASC
6. Get all 1000 names of baby boy and sort them ascending :
https://everydaysmarter.herokuapp.com/api/babyNames/boy?sort=ASC
7. Get all 1000 names of baby girl and sort them descending:
https://everydaysmarter.herokuapp.com/api/babyNames/girl?sort=DESC
8. Get all 1000 names of baby boy and sort them descending:
https://everydaysmarter.herokuapp.com/api/babyNames/boy?sort=DESC
